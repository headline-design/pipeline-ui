# Card

Card renders a responsive box-model layout component. Inherits props from Box component.

import {Card, Text, Icon, Button} from 'pipeline-ui'

<Card bg={"grey"} width={"auto"} maxWidth={"420px"} mx={"auto"} px={[3, 3, 4]}>
  <Text
    caps
    fontSize={0}
    fontWeight={4}
    mb={3}
    display={"flex"}
    alignItems={"center"}
  >
  <Icon name={"AccountBalanceWallet"} mr={2} />
    Connect your Wallet:
  </Text>
  <Button >Check My Balance</Button>
</Card>

## Usage Example

```jsx
import {Card, Text, Icon, AlgoButton, Pipeline} from 'pipeline-ui'
```
