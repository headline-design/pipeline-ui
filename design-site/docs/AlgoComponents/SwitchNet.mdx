# SwitchNet

A checkbox component to switch between MainNet and TestNet. The box is checked by default. Unchecking it will automatically change the default genesis hash and server url. 

:::tip

Before switching nets, make sure to change the net you are using via the MyAlgo Connect interface.

:::
